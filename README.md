# Aeonics Frontend Framework (AFF)
This repository contains the complete AFF template structure and a sample home page.

Please refer to the Aeonics documentation for more details about how to create your web applications easily.

## Layout
* `ae/`: core reusable framework assets (vanilla JS modules and lazy-loaded CSS) that can be shared by multiple websites.
* `demo/`: example site that consumes the shared core via standard ES modules; individual sites live in their own top-level folders.
